Summary

It is well known that the temperatures near the equator are consistently high throughout the year. The purpose of this project is to visualize the temperature difference dependent on a locations latitude. Wind, humidity, and cloud coverage were also visualized so we can see if there is any other factor affected near the equator rather than just the temperature. This data is obtained by creating a query accessing the Open Weather API and randomizing coordinates. Citypy is used which allowed to obtain the nearest city to a randomly selected coordinate.
